Emotion To Text (Cal Hacks 5.0)

Nov 2018

Javascript • Microsoft Azure Vision API • MongoDB • Node.js • Socket.IO

Link: Cal Hacks Submission

Created chat service that allows users to communicate with others using text generated by a snapshot of their face. The user is able to take a photo of their face using their webcam which generates a text response depending on the user's emotion (ie. happiness, sadness, surprise, anger, etc.). The user is then able to send this generated text reply, instead of typing out a response themselves.
