Mineyeeter

April 2018

A-Frame • CSS • Firebase Realtime Database • Github Pages • HTML • Javascript

Link: Demo

Web VR twist on a classic game - Minesweeper - by combining physicality of VR along with a  multiplayer aspect, which is unique to typical VR games. VR player is in the minefield and must navigate the grid with the help of the non-VR player who must figure out where the mines are.
